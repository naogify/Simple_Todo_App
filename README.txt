Simple todo
==== 
###Overview
This is the simple todo app. It uses Localstorage api of html5. 

###Description
*テキストエリアに文章を入力、右の日付入力欄で期限を設定する。
*追加を押すと、予定がリストに追加される。全てを削除で、予定を全て削除。
*htmlタグを入力するとアラートを出した後、文字化して表示。
*一応、タスクが済むとチェックボックスから完了の印をつけれるが、新たに予定をポストすると
元に戻ってしまう
